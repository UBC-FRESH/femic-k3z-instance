Figure Appendix
===============
This appendix catalogs the current K3Z figure set under ``plots/`` and provides
stable filenames for classroom references and release QA.

Core Teaching Figures
---------------------

.. list-table::
   :header-rows: 1

   * - Figure
     - Path
     - Caption
   * - Analysis area strata distribution
     - ``plots/strata-tsak3z.png``
     - Strata composition and area distribution for the K3Z base case.
   * - Analysis area map
     - ``docs/_static/k3z_analysis_area_map.png``
     - Fragment geometry map colored by AU.
   * - VDYP low/med/high envelopes
     - ``plots/vdyp_lmh_tsak3z-*.png``
     - Fitted VDYP envelopes by stratum and SI class.
   * - VDYP fit diagnostics
     - ``plots/vdyp_fitdiag_tsak3z-*.png``
     - Observed-vs-fit checks for each stratum and SI band.
   * - TIPSY vs natural curve overlays
     - ``plots/tipsy_vdyp_tsak3z-*.png``
     - Per-AU overlays comparing regenerated curves and baseline natural curves.
